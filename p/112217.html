热血赛马娘：《赛马娘 热血喧闹大感谢祭》评测

Gamer Daily

2024-09-06 15:45:32

从《赛马娘 热血喧闹大感谢祭》的两个开发商Cygames和Arc System Works一起合作开发也能一窥本作不管是从玩法和IP都十分的粉丝向。作为赛马娘手游发行的Cygames想必不用介绍了，但如果你不知道Arc System Works，那么我提一个热血系列，想必你也豁然开朗了。


这么多年兜兜转转主机也都换了好几个世代了，Arc System Works对待热血系列的正统续作依旧不怎么上心，反倒是炒冷饭的重制和番外做得起劲。日本的游戏开发者偏保守的特性，使得他们在开发正作时不敢过多的加入新变革玩法，相反在番外时又特别乐意进行尝试，这点就包括了本作《赛马娘 热血喧闹大感谢祭》。当然主要指的还是Arc System Works做的热血系列玩法。



《赛马娘 热血喧闹大感谢祭》虽然顶着赛马娘IP，但玩法和手游版并不一样，更偏向是聚会型游戏。本体游戏的玩家开始可选四支马娘队伍，有超过二十名马娘角色在游戏中出场，而且每支队伍中的每名马娘都有各自不同的数值擅长和自带技能。这个选择参赛队伍界面是不是一股热血新纪录的既视感来了。



本作的玩法模式主要分为了故事剧情、本地自定义对战和线上联机。故事剧情模式的玩法基本上也是和本地对战的小游戏一样，而本地对战包括了四个对战小游戏，“粉丝大感谢祭 大障碍赛”、“篮球争夺竞标赛”、“赛马娘躲避球冠军赛”和“大胃王打比”构成，这四个对战小游戏组成了本作的主要游玩内容，而通过游玩这些对战小游戏或者通关故事剧情的关卡，玩家可以利用回馈的奖励来定制自己的赛马娘队伍，以此作为游玩的动力。



而关于上面提到的四个对战小游戏，玩法对于热血系列玩家来说就一点也不陌生了。“赛马娘躲避球冠军赛”就是字面意思的躲避球了，规则很简单，场外NPC会及时对场内发放皮球，玩家需要抢夺皮球后，投掷击中其他玩家，并再次捡起皮球投掷，以此往复，获得最终胜利。而不同于现实规则一击中就会出局的设定不同，本作增加了体力槽以更贴合游戏设计，即使被皮球击中多次，只要体力槽没被清空，该回合也不会被淘汰出局。


但如果只是相互投掷皮球，那么游戏未免也太无聊了，于是游戏也结合了赛马娘的主题，之前我们有提到，游戏中的每一名马娘都有自带技能，这些技能普遍具有无法防御或者大范围伤害的属性，尽管发动技能尽管需要蓄力时间，可一旦发动，就能扭转战局。除技能之外，玩家也可以进行简单的蓄力扔球，或者瞬时接球，这些也能触发各种特效给游戏带来画面视觉升级。


这使得比起热血躲避球的枯燥玩法，游戏从单纯的拿着球就投出去，变成了斗智斗勇激烈的角逐战。



篮球争夺锦标赛的规则就更贴近热血篮球的玩法了。最多可以四名玩家在场，对篮球场两边的篮筐投球，得分多者胜，而且投篮边的篮筐也不止一个，而是会出现多个篮筐，一次性投入的篮筐越多得分也自然越多了。


而玩家不光自己持球，也可以对其他持球玩家进行抢夺篮球，手持篮球时也可以进行闪避，并且也可以在投篮时蓄力，从而提高进球概率，而如果在对方准备投篮时显示出的篮球运动轨迹线上跳跃，即可打断对方的投篮。


游戏当然也不会只有这么简单的设计，随着比赛时间的推进，现场环境也会进行实时变化，比如同一边相距一定距离的多个篮筐会逐渐移动到整齐的纵列，这样一次投球更容易进入多次篮筐得分超级加倍，篮球场上也会开始出现传送带和障碍物来妨碍玩家的运球和投球动作，这也使得球场上的扭转变数更多，乐趣更佳。



“大胃王打比”的游戏规则就相对来说没那么激烈了，而且操作也十分简单上手，规则简单来讲，玩家要先选择两名马娘参赛，一方马娘负责传递餐盘食物，另一个马娘负责吃，谁吃的越多光盘越多自然就获胜了。


玩家要操作负责传递的马娘，将NPC送到场内的食物，端到队友的餐桌上，而负责吃的马娘，玩家需要负责技能释放，来增加进食速度。由于每次传递入场的食物都是有足够多的，所以玩家之间并不需要通过抢夺食物而增加己方胜率，想要加快吃饭的速度，就需要对场上的食物进行颜色区分。每种餐盘上的食物都有各自的颜色，玩家必须按照相同的颜色进行正确的摆盘，方能提高吃饭马娘的速度。


虽说玩家之间不用抢夺食物，全凭自己手速，但作为聚会游戏，肯定还是有一点使绊子的设定的，比如说随着进食也会累积技能槽，有的马娘角色技能就是击晕其他玩家马娘，来延缓他人进度，或者加快己方的移动速度，这些都是让玩家制胜的小技巧。



以上就是对本作主要玩法的简单介绍，尽管本作的玩法与赛马娘手游玩法相去甚远，但手游玩家谁不是奔着立绘去的呢，更何况作为一款聚会的合家欢游戏来说，本作也是十分优秀的。


不光是在套用了赛马娘的IP，也结合了热血系列中几款设计优秀的运动玩法。每一场的比赛都存在能让玩家逆势翻盘的惊喜，就连最后当一项赛事结束后，像热血新纪录一样进入到总评分环节，除了每一场比赛的胜出马娘，还有对某场比赛中表现特别优异的马娘个人表彰加分，即使你在比赛中的评分落后，在最终评分时，玩家都有逆转翻盘的意料之外出现，这给聚会中的玩家带来了无限惊喜和乐趣。
